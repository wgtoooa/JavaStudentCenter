# EduManager

Мини-проект для изучения основ объектно-ориентированного программирования в Java.
Проект моделирует работу простого учебного центра: студентов, курсы и сервисы для взаимодействия между ними.

## Структура проекта

```
src/
 ├─ app/            // Точка входа программы
 │   └─ Main.java
 ├─ model/          // Классы сущностей
 │   ├─ Student.java
 │   └─ Course.java
 ├─ service/        // Логика работы с объектами
 │   └─ StudentService.java
 └─ util/           // Вспомогательные утилиты
     └─ MathUtil.java
```

## Компоненты

### Student
Модель студента.
Поля: имя, возраст, рейтинг.
Функциональность: валидация, вывод информации, проверка совершеннолетия.

### Course
Модель курса.
Поля: название, часы обучения, список студентов.
Функциональность: назначение студентов, вычисление среднего рейтинга, подсчёт количества курсов.

### StudentService
Сервис для работы со студентами.
Функциональность: изменение рейтинга, операции над массивами студентов.

### MathUtil
Утилитарный класс со статическими методами.
Содержит перегруженные методы суммирования, рекурсивный факториал и счётчик вызовов.

### Main
Точка входа.
Демонстрирует взаимодействие всех компонентов проекта.

## Цель проекта

Закрепление базовых тем Java:
- создание и использование классов
- конструкторы и перегрузка
- инкапсуляция и модификаторы доступа
- работа с объектами и массивами объектов
- статические методы и поля
- рекурсия
- структура проектов и работа с пакетами
